# Soil Toxic Metal Pollution and Kidney Function

This repository archives the data tables and analysis scripts for the manuscript on residential soil metal pollution and kidney function in middle-aged and older Chinese adults.

## Submission package

The current prepared archive is stored under:

- `Ecotoxicology_and_Environmental_Safety/Data_and_Code/`

## Data and code contents

The archive includes the cleaned analysis dataset, reanalysis summary tables, mixture-analysis outputs, an updated baseline table, and scripts used to regenerate key manuscript outputs.

## Notes

The current Table 1 uses quartiles of an overall residential soil metal mixture score. The score was calculated as the sum of metal-specific quartile scores for As, Cd, Co, Cr, Cu, Ni, and Pb in the complete-case descriptive sample.
